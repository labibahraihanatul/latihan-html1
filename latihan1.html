HTML atau Hyertext Markup Languange adalah bahasa markah yang digunakan untuk membuat sebuah halamamn web dan memberitahu browser bagaimana menampilkan konten berupa taks, gambar, tabel, form, hyperlink, dll.

CSS adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dokumen. Ada dua sifat CSS yaitu internal dan eksternal. CSS internal dimasukkan secara langsung kedokumen web yang akan didesain.
